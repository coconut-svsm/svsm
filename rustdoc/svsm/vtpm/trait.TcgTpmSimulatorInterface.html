Expand description

This implements one handler for each TpmPlatformCommand supported by the VTPM Protocol. These handlers are based on the TPM Simulator interface provided by the TPM 2.0 Reference Implementation, but with a few changes to make it more Rust idiomatic.

fn signal_nvon(&self) -> Result<(), SvsmReqError>

In a system where the NV memory used by the TPM is not within the TPM, the NV may not always be available. This function indicates that NV is available.
