fn update_apic_base(&self, and_mask: u64, or_mask: u64)

Updates the APIC_BASE MSR by reading the current value, applying the and_mask, then the or_mask, and writing back the new value.

§Arguments

and_mask - Value to bitwise AND with the current value. or_mask - Value to bitwise OR with the current value, after and_mask has been applied.
