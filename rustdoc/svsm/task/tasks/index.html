KernelThreadStartParameter
This trait is used to describe a type that can be used as the start parameter for a kernel thread. The thread start path requires that all start paremeters be able to fit into a single register, which constrains them to usize. Other types can be used as long as there is an appropriate mechanism to covert those types to and from the usize that must be passed via the parameter register. This trait defines such a conversion mechanism.
