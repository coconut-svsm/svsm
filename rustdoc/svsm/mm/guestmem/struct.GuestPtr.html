pub unsafe fn try_read(&self) -> Result<T, SvsmError>
where T: TryFromBytes,

Attempts to read the T behind the pointer, verifying that it has a valid representation in the process. This may be used for types for which not all bit patterns are valid.

§Safety

See safety considerations for GuestPtr::read().

§Errors
  • If the access caused a fault, this returns Err(SvsmError::Fault).
  • If the read data did not have a valid format for T, this returns Err(SvsmError::Mem).
