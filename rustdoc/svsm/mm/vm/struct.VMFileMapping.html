pub fn new( file: &FileHandle, offset: usize, size: usize, flags: VMFileMappingFlags, ) -> Result<Self, SvsmError>

Create a new [‘VMFileMapping’] for a file. The file provides the backing pages for the file contents.

§Arguments

  • ‘file’ - The file to create the mapping for. This instance keeps a reference to the file until it is dropped.

  • ‘offset’ - The offset from the start of the file to map. This must be align to PAGE_SIZE.

  • ‘size’ - The number of bytes to map starting from the offset. This must be a multiple of PAGE_SIZE.

§Returns

Initialized mapping on success, Err(SvsmError::Mem) on error
