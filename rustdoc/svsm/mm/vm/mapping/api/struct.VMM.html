Expand description

A single mapping of virtual memory in a virtual memory range

Fields§

§link: AtomicLink

Link for storing this instance in an RBTree

§range: Range<usize>

The virtual memory range covered by this mapping It is stored in a RefCell to check borrowing rules at runtime. This is safe as any modification to range is protected by a lock in the parent data structure. This is required because changes here also need changes in the parent data structure.

§mapping: Arc<Mapping>

Pointer to the actual mapping It is protected by an RWLock to serialize concurent accesses.
