Struct VMalloc

pub struct VMalloc {
    alloc: RawAllocMapping,
    flags: PTEntryFlags,
}
Expand description

Virtual mapping backed by allocated pages. This can be used for memory allocation if there is no need for the memory to be physically contiguous.

This is a wrapper around RawAllocMapping.

Fields§

§alloc: RawAllocMapping

RawAllocMapping used for memory allocation

§flags: PTEntryFlags

Page-table flags to map pages
