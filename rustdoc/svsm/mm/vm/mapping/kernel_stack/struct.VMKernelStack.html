Struct VMKernelStack 

Source 
pub struct VMKernelStack {
    alloc: RawAllocMapping,
    guard_pages: usize,
    shadow: bool,
}
Expand description

Mapping to be used as a kernel stack. This maps a stack including guard pages at the top and bottom.

Fields§

§alloc: RawAllocMapping

Allocation for stack pages

§guard_pages: usize

Number of guard pages to reserve address space for

§shadow: bool

True for shadow stacks
